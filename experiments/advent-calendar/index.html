How to use this?

This is a customizable advent calendar for the year of 2021. It is an experimental project, so it may not be the best it can be. If you have any suggestions, please let me know in the discord: #advent-calendar.
To get started with the calendar, please put some items in the "Items" section. We reccomend at least five items, but we support any amount. After that, click the button below to generate each day. You can only generate each day once, so be careful.
Just in case you make a mistake, well you have a problem. Mistake support is coming soon. This project will update once a year, for every advent calendar. The next update will be in 2022. A couple style changes might be made this year, but no real changes will be made.
